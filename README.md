# Orcinus

금융 투자를 위한 올인원 인프라 플랫폼

## 🚀 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 📁 프로젝트 구조

```
orcinus/
├── public/
│   └── favicon.svg
├── src/
│   ├── App.jsx          # 메인 랜딩페이지
│   ├── main.jsx         # React 엔트리
│   └── index.css        # Tailwind CSS
├── index.html
├── package.json
├── vite.config.js
├── tailwind.config.js
└── postcss.config.js
```
